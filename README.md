# Cloudflare Feedback Prototype

A prototype for aggregating and analysing product feedback using Cloudflare Workers using AI.

## Overview
This project explores how product feedback from multiple sources can be aggregated.

## Features
- Collects feedback from multiple sources (GitHub, Discord, Support Tickets, etc.)
- Theme extraction and clustering
- Urgency classification

## Tech Stack
- **Cloudflare Workers**: Serverless application hosting
- **D1 database**: Feedback storage and querying
- **Workers AI**: AI-powered analysis and insights
